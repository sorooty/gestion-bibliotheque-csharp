# Gestion Bibliotheque - C# .NET

Project developed as part of the engineering program at CESI Paris.
Objective: apply the core principles of **Object-Oriented Programming** in C#.

## Project Structure

| Class | Role |
|---|---|
| `Livre` | Represents a book with title, author and availability status |
| `Bibliotheque` | Manages a collection of books (add, display) |

## Features

- Book creation with a parameterized constructor
- Adding books to a library
- Displaying book information via `ToString()`

## Technologies

- Language: C# (.NET)
- IDE: Visual Studio

## Getting Started

```bash
dotnet run
```

## Concepts Covered

- Classes and objects
- Constructors
- Encapsulation (`private`, `public`)
- `List<T>`
- Method overriding (`ToString()`)
