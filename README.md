# GestionBibliotheque

---

## 🇫🇷 Français

Application console en C# permettant de gérer une bibliothèque de livres.

**Fonctionnalités :**
- Ajouter, supprimer et rechercher un livre
- Emprunter et retourner un livre
- Afficher la liste complète des livres

**Technologies :** C# / .NET · Application console

---

## 🇬🇧 English

A C# console application to manage a book library.

**Features:**
- Add, delete, and search for a book
- Borrow and return a book
- Display the full list of books

**Tech stack:** C# / .NET · Console application
